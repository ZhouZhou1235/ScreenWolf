### 描述
让可爱的宠物们待在电脑屏幕上~ 驱散枯燥和无聊-w-
屏幕有狼是一款基于 Java Swing 的桌面宠物游戏，
玩家能在桌面上放置可交互的虚拟宠物。
游戏支持加载新宠物，
每只宠物拥有独立的动画、数值和行为逻辑。
也就是说，所有宠物均可自定义！


### 旧版说明
